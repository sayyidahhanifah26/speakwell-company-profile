# SpeakWell

Website pembelajaran bahasa Inggris dengan React + TypeScript + Vite.

## Tech Stack
- React, TypeScript, Vite, Tailwind CSS
- Zustand (state management)
- Backendless (backend/auth)
- RandomUser API (teams data)

## Setup
```bash
npm install
npm run dev
```

## Features
- Home, About, Services, Teams, Blog
- Authentication (Backendless)
- CRUD Blog (protected route)
- Responsive design
